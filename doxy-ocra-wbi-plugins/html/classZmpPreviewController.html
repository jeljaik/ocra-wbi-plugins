Detailed Description

Implementes an extended ZMP preview controller as an unconstrained QP problem.

Note
Put original ZMP preview control reference along with Aurelien's
Author
Jorhabib Eljaik

Given a rough ZMP trajectory and a corresponding consistent CoM velocity, this class computes at a fast rate optimal CoM jerks over a refined preview horizon.

1 /*
2 
3  The quadprog_solve() function implements the algorithm of Goldfarb and Idnani
4  for the solution of a (convex) Quadratic Programming problem
5  by means of an active-set dual method.
6 
7 The problem is in the form:
8 
9 min 0.5 * x G x + g0 x
10 s.t.
11  CE^T x + ce0 = 0
12  CI^T x + ci0 >= 0
13 
14  The matrix and vectors dimensions are as follows:
15  G: n * n
16  g0: n
17 
18  CE: n * p
19  ce0: p
20 
21  CI: n * m
22  ci0: m
23 
24  x: n
25 
26  The function will return the cost of the solution written in the x vector or
27  std::numeric_limits::infinity() if the problem is infeasible. In the latter case
28  the value of the x vector is not correct.
29 
30  References: D. Goldfarb, A. Idnani. A numerically stable dual method for solving
31  strictly convex quadratic programs. Mathematical Programming 27 (1983) pp. 1-33.
32 
33  Notes:
34  1. pay attention in setting up the vectors ce0 and ci0.
35  If the constraints of your problem are specified in the form
36  A^T x = b and C^T x >= d, then you should set ce0 = -b and ci0 = -d.
37  2. The matrix G is modified within the function since it is used to compute
38  the G = L^T L cholesky factorization for further computations inside the function.
39  If you need the original matrix G you should make a copy of it and pass the copy
40  to the function.
